## Development

### Linting and Formatting

This project uses [Biome](https://biomejs.dev/) for linting and formatting.

- **Format**: Run `npm run format` to format code.
- **Lint**: Run `npm run lint` to check for linting errors.
- **Check**: Run `npm run check` to check for linting errors, format code and organize imports.

To automatically fix formatting, linting errors and organize imports, run:

```bash
npm run check:fix
```
