## Development

### Linting and Formatting

This project uses [Biome](https://biomejs.dev/) for linting and formatting.

- **Lint**: Run `npx @biomejs/biome lint` to check for linting errors.
- **Format**: Run `npx @biomejs/biome format` to see formatted code.

To automatically fix linting errors and format files, you can run:

```bash
npx @biomejs/biome check --write .
```
